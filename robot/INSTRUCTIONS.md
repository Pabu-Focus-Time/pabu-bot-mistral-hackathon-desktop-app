To run Mujoco, you need to create a virtual environment

You need Daemon: mjpython -m reachy_mini.daemon.app.main --sim
And running Reachy Mini SDK: SIMULATION_MODE=false python3 app.py

Documentation:

- HuggingFace Reachy Mini Python SDK: https://huggingface.co/docs/reachy_mini/SDK/python-sdk
- Reachy Mini Architecture: https://huggingface.co/docs/reachy_mini/SDK/media-architecture
- Reachy Mini Wireless: https://huggingface.co/docs/reachy_mini/platforms/reachy_mini/get_started
